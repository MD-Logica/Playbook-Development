# PLAYBOOKMD — PHASE 1A DESIGN REFINEMENT

---

## CONTEXT

The app shell and navigation have been built but the visual design is generic and does not match the PlaybookMD brand. This prompt is a surgical redesign of the existing layout — do not rebuild the structure or change any functionality. Only update the visual design, color system, typography, spacing, and microcopy. All navigation links, Clerk auth, Prisma data fetching, and routing must remain intact.

---

## THE PROBLEM WITH THE CURRENT DESIGN

The current UI looks like a generic AI-generated SaaS template. Every element has equal visual weight, nothing feels intentional, and there is no brand personality. It does not reflect PlaybookMD's identity: a premium, confident, modern platform that makes legacy EMRs look obsolete. That gap needs to close completely in this phase.

---

## BRAND IDENTITY — INTERNALIZE THIS BEFORE TOUCHING ANY CODE

PlaybookMD is **not** a medical software company. It is a revenue intelligence platform that happens to serve medical practices. The brand voice is confident, sharp, and irreverent toward legacy systems. The product should feel like it was built by people who think the entire EMR industry is a joke — and then actually built something better.

**The visual target:** Linear.app met with a premium healthcare aesthetic. Dark, structured, precise. Every pixel intentional. Nothing that screams "medical software."

---

## DESIGN DIRECTION

### Overall Layout
- **Dark sidebar** (`#0A1628` — deep navy-black), white content area
- This creates immediate visual separation and makes the emerald brand color pop
- The sidebar is where the brand lives; the content area is where work happens

### Color System — Replace ALL existing colors with these

Define as CSS custom properties in `app/globals.css`. Remove all previous color variables.

```css
:root {
  /* Brand */
  --brand-primary: #10B981;        /* Emerald 500 — primary actions, active states */
  --brand-secondary: #059669;      /* Emerald 600 — hover states */
  --brand-deep: #064E3B;           /* Emerald 900 — dark accents, gradient end */
  --brand-light: #D1FAE5;          /* Emerald 100 — subtle tints */
  --brand-glow: rgba(16,185,129,0.15); /* Emerald glow for active states */

  /* Sidebar (dark) */
  --sidebar-bg: #0A1628;           /* Deep navy-black */
  --sidebar-border: rgba(255,255,255,0.06);
  --sidebar-text: rgba(255,255,255,0.65);
  --sidebar-text-active: #FFFFFF;
  --sidebar-item-hover: rgba(255,255,255,0.06);
  --sidebar-item-active: rgba(16,185,129,0.15);
  --sidebar-icon-default: rgba(255,255,255,0.4);
  --sidebar-icon-active: #10B981;
  --sidebar-divider: rgba(255,255,255,0.08);

  /* Content area (light) */
  --bg-primary: #FFFFFF;
  --bg-secondary: #F9FAFB;
  --bg-tertiary: #F3F4F6;

  /* Text */
  --text-primary: #111827;         /* Near black — headings */
  --text-secondary: #374151;       /* Dark gray — body */
  --text-muted: #6B7280;           /* Medium gray — metadata */
  --text-subtle: #9CA3AF;          /* Light gray — placeholders */

  /* Borders */
  --border-default: #E5E7EB;
  --border-strong: #D1D5DB;

  /* Semantic */
  --success: #10B981;
  --warning: #F59E0B;
  --danger: #EF4444;
  --info: #3B82F6;

  /* Top bar */
  --topbar-bg: #FFFFFF;
  --topbar-border: #E5E7EB;
}
```

### Typography
- **Font:** `Geist` (already in Next.js) — use Geist Sans for UI, Geist Mono for any data/code
- Apply globally via `layout.tsx` using Next.js font optimization
- **Scale:**
  - Page titles: 20px, weight 600, color `--text-primary`, letter-spacing -0.3px
  - Section headings: 16px, weight 600, color `--text-primary`
  - Body: 14px, weight 400, color `--text-secondary`
  - Labels/metadata: 13px, weight 500, color `--text-muted`
  - Tiny/captions: 12px, weight 400, color `--text-subtle`
- Never use font-weight below 400

## SIDEBAR REDESIGN

This is the most important visual element. Make it feel premium.

### Logo Area
- Height: 56px (matches top bar)
- Padding: 0 16px
- Display the SVG logo icon inline followed by "PlaybookMD" wordmark
- Logo icon: the three stacked rectangles SVG with emerald gradient (embed directly as inline SVG)
- Wordmark: "PlaybookMD" in white, font-size 15px, font-weight 600, letter-spacing -0.2px
- The "MD" portion can be slightly lighter weight (400) to create typographic contrast
- Bottom border: `1px solid var(--sidebar-divider)`

### Navigation Items
- Item height: 34px
- Border radius: 6px
- Padding: 0 10px
- Margin: 1px 8px (horizontal margins create floating effect)
- Default state: icon at `var(--sidebar-icon-default)`, label at `var(--sidebar-text)`, no background
- Hover state: background `var(--sidebar-item-hover)`, icon slightly brighter, 150ms ease transition
- Active state: background `var(--sidebar-item-active)`, icon `var(--sidebar-icon-active)` (emerald), label `var(--sidebar-text-active)` (white), no border or heavy indicator — the emerald icon IS the active indicator
- Icon size: 15px
- Label: 13px, font-weight 500
- Gap between icon and label: 9px

### Section Dividers
- `1px solid var(--sidebar-divider)` with 6px vertical margin top and bottom
- No section labels/headers — the groupings are self-evident

### Sidebar Footer
- Separated from nav by a divider
- Padding: 12px 16px
- Practice name: 11px, uppercase, letter-spacing 0.06em, color `rgba(255,255,255,0.35)` — subtle, not dominant
- User name: 13px, weight 500, color `rgba(255,255,255,0.8)`
- Avatar: 26px circle, emerald background (`#059669`), white initials, 11px font
- Sign out: small icon button (`LogOut` from lucide, 14px), `rgba(255,255,255,0.3)` color, hover to `rgba(255,255,255,0.7)`, positioned to the right
