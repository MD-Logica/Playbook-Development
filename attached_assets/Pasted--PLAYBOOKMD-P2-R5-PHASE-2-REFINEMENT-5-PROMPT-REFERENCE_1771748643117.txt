## FIX 1: COLUMN HEADERS — LIGHTEN SIGNIFICANTLY

The current dark navy column headers are too heavy and oppressive. They clash with the card-based content below them and make the board feel like a command interface rather than a work surface.

**Replace the dark header background with a light tinted approach:**

Each column header background should use the column's accent color at **8% opacity** against white. So an emerald-accented column gets `rgba(16, 185, 129, 0.08)` as its header background. A blue-accented column gets `rgba(59, 130, 246, 0.08)`. Etc.

**Column header styling:**
- Background: `rgba([stage accent color], 0.08)` — light tint, not dark
- Border-bottom: `2px solid [stage accent color]` — this is the primary accent, full opacity
- Stage name text: `--text-primary` (dark), 12px, weight 700, uppercase, letter-spacing 0.06em
- Card count badge: small pill, `[stage accent color]` background at 15% opacity, accent color text, 11px
- Stage value ($700, $2,550 etc): Geist Mono, 12px, weight 600, `--text-secondary`, right-aligned
- Header height: 40px, padding 0 14px, display flex, align-items center, justify-content space-between
- Border-radius: 8px 8px 0 0

This creates column identity through color tinting without the visual weight of dark headers.
